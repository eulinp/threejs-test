👩🏽‍🚀 ¿Que es ThreeJS?

Three.js es un marco webGL que se usa ampliamente debido a su facilidad de uso.

Three.js es otra capa de encapsulación basada en la interfaz api de WebGL.

Esta biblioteca fue creada y liberada en GitHub por el español Ricardo Cabello en abril de 2010, mejor conocido como Mr.doob .

Three.js encapsula objetos de uso común en la programación de gráficos 3D de una manera simple e intuitiva, además, utiliza muchas técnicas avanzadas de motores gráficos en desarrollo, lo que mejora enormemente el rendimiento. Debido a muchos objetos incorporados y herramientas fáciles de usar, Three.js es muy poderoso. Finalmente, Three.js es completamente de código abierto, puede encontrar su código fuente en GitHub, y muchas personas contribuyen con el código para ayudar Mr.doob a mantener este marco.

💻 ¿Qué es WebGL?

WebGL (Web Graphics Library) es una API de JavaScript para renderizar gráficos 3D y 2D interactivos en cualquier navegador web compatible sin el uso de complementos.
WebGL se puede usar en elementos HTML5 mediante la introducción de una API que está estrechamente alineada con OpenGL ES 2.0. En resúmen, WebGL es utilizado por los desarrolladores para crear aplicaciones gráficas interactivas en la web.
